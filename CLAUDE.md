## SEO + AI Search (GEO) rules for every page
- Exactly ONE H1 per page.
- A “direct answer” paragraph near the top (2–4 sentences).
- Clear H2/H3 sections, short paragraphs, concrete language.
- FAQ section with 6–10 Q&As per solution page.
- Add JSON-LD schema (Organization, WebSite sitewide; Service + FAQPage on solution pages).
- sitemap.xml + robots.txt + canonical URLs.

## Definition of Done (DoD)
- Build succeeds.
- Pages are pre-rendered / statically generated for SEO.
- Lighthouse-friendly basics: fast loading, optimized images, minimal JS.
- No broken links; nav + footer consistent.
- Forms validate + success state works.

## Working style
- Explore → Plan → Implement → Verify (always provide verification commands or checks).
- Prefer small, reviewable commits.
